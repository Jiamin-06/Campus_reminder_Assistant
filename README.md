# 校园全能提醒小助手 · 项目文件

## 文件结构

1. **widget1.html** — 主助手（6模块标签页界面）
2. **widget2.html** — AI录入节点（6步引导式信息收集）
3. **widget3.html** — 自动处理节点（6条提醒规则 + 固定格式输出）

## 设计规范

- 配色：高校蓝 `#4A90D9` + 活力橙 `#FF6B35` + 警示红 `#E74C3C`
- 字体：`-apple-system, "PingFang SC", "Microsoft YaHei", sans-serif`
- 无后端，纯前端 localStorage（已改为内存数据）
- 响应式，支持手机端

## 固定输出格式模板

```
【XX大学·校园全能提醒小助手】
📅 今日课表：{{课程表信息}}
📝 作业倒计时：{{作业提醒}}
📚 考试倒计时：{{考试提醒}}
📦 快递提醒：{{快递信息}}
💰 余额预警：{{余额预警}}
✅ 今日待办：{{汇总清单}}
```
